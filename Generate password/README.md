# Bash Password Generator

A simple Bash script that generates a secure random password of a user-specified length and saves it to a file.

## Features

- Generates passwords containing letters, numbers, and special characters (`_!@#$%^&*><{}()[]+-`)  
- User specifies the length of the password  
- Saves username and generated password with timestamp to `passwords.txt` in the user's home directory  
- Simple terminal-based script with colored output

## How to Run

1. **Give execution permission** (first time only):
chmod +x password-generator.sh
2. ./password-generator.sh
